# Payroll Incident — Access Control Case Study

This repository contains a concise, portfolio-ready case study describing a payroll incident caused by poor account lifecycle management and excessive privileges.

Files
- `PAYROLL_INCIDENT_CASE_STUDY.md` — Full case study and recommendations.
- `RBAC-matrix.csv` — Sample Role-Based Access Control matrix (roles and permissions).

Usage
- Review the case study and adapt the recommendations to your environment.
- Use the `RBAC-matrix.csv` as a starting point for defining roles and permissions in your IAM/AD system.

Contributions
- Open an issue or submit a PR if you want additional sample policies, automation scripts, or a PDF export added.
